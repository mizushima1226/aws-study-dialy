# 3章 AWSを使うためのツール

## 自由記入

- サービスを管理すための機能
  - マネジメントコンソール：Webブラウザ
  - CLI：マネジメントコンソールよりいろんなことできる
  - IAM：グループ・ポリシーなどで一括管理できる
  - コストマネジメント：コストをグラフなどで確認できる
  - CloudWatch：監視ツール1
- AWSアカウント（rootユーザ）：管理者権限を持ってる
  - なんでもできるから普通はIAM使う

- IAM：各種サービスへのアクセスを管理する
  - IAMユーザ：人に対して与えるIAM
  - IAMロール：サーニスやプログラムなどに与えるIAM
  - IAMグループ：ユーザはIAMグループにまとめることができる。同じ権限を与えたい時、一括管理できる
  - IAMポリシー：実行者（ユーザ・ロール・グループ）がどのサービスにアクセスできるのか決まり事を設定する
    - アイデンティティベースのポリシー：誰（ユーザ・ロール・グループ）が何（起動・停止・読み込み・書き込み・削除）をできるか？
    - リソースベースのポリシー：どのサービス（EC2・S3）が何（起動・停止・読み込み・書き込み・削除）を許可するか？
